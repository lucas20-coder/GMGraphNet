# GMGraphNet

Official implementation of **GMGraphNet: Guidance-Motion Graph Learning for Short-Term Tropical Cyclone Track Forecasting**.

This repository provides the source code, preprocessing scripts, model configurations, and reproduction instructions for the GMGraphNet framework.

## Status

The code is being organized and will be made publicly available upon acceptance of the manuscript.

## Data

The study uses publicly available IBTrACS best-track data and ERA5 reanalysis data. The processed storm-centered samples are not included because the preprocessed ERA5 patch archive is large and memory intensive.
